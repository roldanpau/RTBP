Obtain rotating Delaunay coordinates (l,L,g,G) from rotating cartesian (x,px).

Remarks:
WE ASSUME that input point is on the Poincare section . We can recover the 4 euclidean coordinates (x,y,px,py) using the energy relation.
Parameters:
[in]mumass parameter for the RTBP
[in]Henergy value
[in]zpoint in rotating cartesian coordinates, z=(x,p_x).
[out]Ypoint in rotating Delaunay coordinates, Y=(l,L,g,G).
Returns:
a non-zero error code to indicate an error and 0 to indicate success.
