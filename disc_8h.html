Discretize the linear segment between $p_0$ and $p_1$ into NPOINTS.

Parameters:
[in]p0,p1endpoints of linear segment
[in]nnumber of points in the discretization (n must be >=2)
[out]lOn return of the this function, it holds the discretization of the linear segment, i.e. an array of n 2D points.
Returns:
a non-zero error code to indicate an error and 0 to indicate success.
Precondition:
Caller must make sure that enough space has been allocated for l (2*n doubles)
Remarks:
The points are equally spaced.
The first point coincides with p0. The last point coincides with p1.
