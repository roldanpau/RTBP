Poincare map of RTBP in Delaunay coordinates

Consider the RTBP in rotating coordinates. Let S be the Poincare section SEC1 or SEC2, corresponding to {l=0} or . Let $x$ be a point in the section, and suppose that flow at a point $x$ is transversal to the section. Compute the n-th iterate of the Poincare map $P^n(x)$ of the RTBP. This procedure also computes, as a side product, the integration time to intersect the Poincare section "n" times.

Parameters:
[in]mumass parameter for the RTBP
[in]sectype of Poincare section (sec = SEC1 or SEC2).
[in]cutsnumber of iterates of the Poincare map (cuts=n: n cuts with the Poincare section). This must be a positive (or zero) integer.
[in,out]xInitial point, 4 coordinates: (l, L, g, G). On return of the this function, it holds the image point $P^n(x)$.
[out]tiOn return, it holds the integration time to intersect the Poincare section "n" times.
Returns:
Returns a non-zero error code to indicate an error and 0 to indicate success. If an integration error is encountered, the function returns a non-zero value.
Remarks:
To compute the inverse Poincare map (i.e., negative iterates of the Poincare map), see prtbp_del_inv.
On successful return of this function, the point $x$ is exactly on the section, i.e. we set coordinate $l$ exactly equal to zero/pi.
