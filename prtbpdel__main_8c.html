Poincare map of RTBP in Delaunay coordinates: main prog.

Let "sec" be a Poincare section in the RTBP, where

  • sec = SEC1 means section {l=0}
  • sec = SEC2 means section {l=pi}. This program computes the n-th iterate of the Poincare map, $P^n(x)$.

1. Input params (stdin):

  • mass parameter
  • type of section "sec"
  • number of iterates "n"

2. for each initial point "x" from stdin, do 2.1. Compute n-th iterate of the Poincare map, $P^n(x)$. 2.2. Output final point $P^n(x)$ and integration time to stdout.

Remarks:
We do not check (and thus we do not impose) that the initial point "x" is on the Poincare section.
