Compute the n-th iterate $P^n$ of the 2D Poincare map:

in the RTBP.

Consider the RTBP in rotating coordinates. Let S be the Poincare section SEC1 or SEC2, corresponding to {y=0, v_y>0} or {y=0, v_y<0}. Fix the Hamiltonian to a given value "H". Using this energy condition, we can work with only two variables, $(x,p_x)$. The third variable $y$ is 0 since we look at the Poincare section, and the fourth variable $p_y$ can be obtained from the energy condition. This function computes the n-th iterate $P^n$ of the 2D Poincare map:

This procedure also computes, as a side product, the integration time to intersect the Poincare section "n" times.

Parameters:
[in]mumass parameter for the RTBP
[in]sectype of Poincare section (sec = SEC1 or SEC2).
[in]Henergy value
[in]cutsnumber of iterates of the Poincare map (cuts=n: n cuts with the Poincare section).
[in,out]pInitial point, 2 coordinates: p=(x,p_x). On return of the this function, it holds the image point $P^n(p)$.
[out]tiOn return, it holds the integration time to intersect the Poincare section "n" times.
Returns:
Returns a non-zero error code to indicate an error and 0 to indicate success.
