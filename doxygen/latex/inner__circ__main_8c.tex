Inner Map of the Circular Problem: main prog.

This program computes the shift \$T\_\-0\$ of the inner map of the circular problem, given by the integral

\[ 2\pi + T_0 = \int_0^{6\pi} \frac{1}{L^{-3}+\mu\partial_L \Delta H_{circ}(\lambda_I^1(s))} ds, \]

where $\gamma(s)$ is the periodic trajectory in the level of energy H, or equivalently

\[ T_0 = \int_{0}^{6\pi} \frac{1}{\dot \ell (\lambda_I^1(s))} ds - 2\pi. \]

Morevover, this program also computes the partial shifts \$\{in\}$^\wedge$\{f,b\}\$ that we will need when computing the outer map:

\begin{eqnarray*} \omega_{in}^f(I) &= \int_0^{4\pi} f0(\lambda_I^2(s)) ds. \\ \omega_{in}^b(I) &= \int_0^{2\pi} f0(\lambda_I^1(s)) ds. \end{eqnarray*}

(See notes \char`\"{}Kirkwood Gaps\char`\"{} by Marcel).

It reads the following input from stdin:


\begin{DoxyItemize}
\item mu mass parameter for the RTBP
\item bInner (flag) bInner == 0 means OUTER/LOWER splitting = \char`\"{}b\char`\"{}, bInner == 1 means INNER/UPPER splitting = \char`\"{}f\char`\"{}.
\end{DoxyItemize}

Then it reads a sequence of input lines from stdin:
\begin{DoxyItemize}
\item p\mbox{[}DIM\mbox{]} periodic point, 4 coordinates: (l,L,g,G). Notice that \$p=p\_\-2\$ corresponds to INNER/UPPER separatrix, \$p=p\_\-1\$ to OUTER/LOWER separatrix.
\end{DoxyItemize}

For each input line, it writes the following output in stdout:
\begin{DoxyItemize}
\item \$w\_\-\{in\}$^\wedge$$\ast$\$ (where $\ast$=f,b depending on bInner flag).
\end{DoxyItemize}

\begin{DoxyRemark}{Remarks}
The integrand is evaluated at an equispaced sequence of points t(0,6). 
\end{DoxyRemark}
