Flow of the RTBP in Delaunay coords.

Compute the flow \$(t,x)\$ of the RTBP for a given time and initial condition. The time \$t\$ may be positive or negative, allowing for forward or backward integration. The trajectory is integrated numerically.


\begin{DoxyParams}[1]{Parameters}
\mbox{\tt in}  & {\em mu} & mass parameter for the RTBP\\
\hline
\mbox{\tt in}  & {\em t1} & integration time\\
\hline
\mbox{\tt in,out}  & {\em x\mbox{[}DIM\mbox{]}} & Initial condition, 4 coordinates: (l, L, g, G). On return of this function, it holds the final point \$(t,x)\$.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
a non-\/zero error code to indicate an error and 0 to indicate success. If an integration error is encountered, the function returns a non-\/zero value.
\end{DoxyReturn}
\begin{DoxyRemark}{Remarks}
A Runge-\/Kutta Prince-\/Dormand (8,9) method is used to solve the ODE. We request absolute local error 10$^\wedge$\{-\/16\} and relative local error 0. 
\end{DoxyRemark}
