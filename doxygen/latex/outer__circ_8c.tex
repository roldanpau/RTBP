Outer Map of the Circular Problem: main prog.

This program computes the integrals $ \omega_{+,-}^* $ related to the outer map of the circular problem. (See notes \char`\"{}Kirkwood Gaps\char`\"{} by Marcel).

It reads the following input from stdin:
\begin{DoxyItemize}
\item mu mass parameter for the RTBP
\end{DoxyItemize}

And a sequence of lines:
\begin{DoxyItemize}
\item T period of periodic orbit.
\item zu preimage of homoclinic point z (in the original flow): P$^\wedge$\{M\}(z\_\-u) = z.
\item M number of poincare iterates to reach z from z\_\-u.
\end{DoxyItemize}

For each input line, it outputs result to stdout:
\begin{DoxyItemize}
\item omega\_\-neg 
\end{DoxyItemize}

Given an energy level $H$, compute $\omega_-^*(H)$.

Given an energy level $H$, compute $\omega_-^*(H)$, defined as

\[ \omega_-^*(I) = lim_{N\to -\infty} \left( \int_0^{6N\pi} f0(\gamma^*(s)) ds + NT_0(I) \right), \]

where $\gamma^*(s)$ is the homoclinic trajectory of the reduced flow that starts at the primary homoclinic point z.

Equivalently, we compute it as

\[\omega_-^*(I) = lim_{N\to \infty} \left( \int_{6N\pi}^0 f0(\gamma^*(s)) ds - NT_0(I) \right), \]

where $\gamma^*(s)$ is the homoclinic trajectory that starts at the point $ z^u = \mathcal{P}^{-N}(z) $. Notice that z$^\wedge$u is in the UNSTABLE manifold of the reduced flow.

Equivalently, we split the integral from 6N to 0 into N parts of size 6:

\[\omega_-^*(I) = lim_{N\to \infty} \left( \sum_{i=N,1} (\int_{6\pi}^{0} f0(\gamma_i(s)) ds - T_0(I)) \right), \]

where $\gamma_i(s)$ is the homoclinic trajectory that starts at the point P$^\wedge$\{N-\/i\}(z$^\wedge$u) = P$^\wedge$\{-\/i\}(z).

This is computed using numerical integration.
