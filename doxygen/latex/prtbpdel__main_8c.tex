Poincare map of RTBP in Delaunay coordinates: main prog.

Let \char`\"{}sec\char`\"{} be a Poincare section in the RTBP, where
\begin{DoxyItemize}
\item sec = SEC1 means section \{l=0\}
\item sec = SEC2 means section \{l=pi\}. This program computes the n-\/th iterate of the Poincare map, \$P$^\wedge$n(x)\$.
\end{DoxyItemize}

1. Input params (stdin):


\begin{DoxyItemize}
\item mass parameter
\item type of section \char`\"{}sec\char`\"{}
\item number of iterates \char`\"{}n\char`\"{}
\end{DoxyItemize}

2. for each initial point \char`\"{}x\char`\"{} from stdin, do 2.1. Compute n-\/th iterate of the Poincare map, \$P$^\wedge$n(x)\$. 2.2. Output final point \$P$^\wedge$n(x)\$ and integration time to stdout.

\begin{DoxyRemark}{Remarks}
We do not check (and thus we do not impose) that the initial point \char`\"{}x\char`\"{} is on the Poincare section. 
\end{DoxyRemark}
