Flow of the Reduced Restricted Three Body Problem

Compute the flow $(s,x)$ of the reduced RTBP for a given time and initial condition. Reduced means that we identify $l$ with time and then we do not need to integrate this variable. The time $s$ may be positive or negative, allowing for forward or backward integration. The trajectory is integrated numerically.

Parameters:
[in]mumass parameter for the RTBP
[in]s1integration time
[in,out]x[DIMRED]Initial condition, 6 coordinates: (l, L, g, G, t, I). On return of this function, it holds the final point $(s,x)$.
Returns:
a non-zero error code to indicate an error and 0 to indicate success. If an integration error is encountered, the function returns a non-zero value.
Remarks:
A Runge-Kutta Prince-Dormand (8,9) method is used to solve the ODE. We request absolute local error 10^{-16} and relative local error 0.
